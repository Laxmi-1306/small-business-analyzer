Small Business Sales & Profit Analyzer is a sleek Streamlit web app that empowers small business owners to effortlessly manage operations. Track sales, expenses, and inventory, generate detailed reports, and forecast future sales with Prophet and interactive Plotly charts—all in one place. It features secure login/registration, multi-business management, transaction categorization, inventory control, and CSV/Excel uploads for advanced analytics. Admins can monitor total users, businesses, and transactions. Built with Streamlit, Pandas, Plotly, Prophet, and SQLite, it’s easy to set up locally (streamlit run app.py) or deploy instantly on Streamlit Cloud
. Future-ready for cloud databases, role-based access, interactive reporting, and notifications, this app turns business data into actionable insights with minimal effort.
